# khtSaveString

---

网页 保存数据到原生 以 key value 保存

## 使用指南

```js
import * as TPTJS from 'tpt-js-sdk'
TPTJS.khtSaveString('khtwebview_invitesPage_refresh', '111');
```

## khtSaveString 参数

| 参数   | 说明   | 类型    | 可选值 | 默认值   |
| ---   | ---- | ------- | ------- | ------ |
| key  | 健值 | string | 需要以`khtwebview_`开头 | - |
| value  | 需要保存的内容 | String | - | - |
